## 3) تحليل كل صفحة

- **`#hr` dashboard:** hero + KPIs (موظفون، غياب اليوم، إجازات معلقة، رواتب).
- **`#hr_employees`:** جدول + فلاتر (فرع، قسم، منصب).
- **`#hr_employee/:id`:** `.hr-emp-hero` variant + tabs (بيانات، حضور، إجازات، راتب، مستندات — Documents cross-module).
- **`#hr_organization`:** شجرة أقسام + مناصب.
- **`#hr_attendance`:** تقويم شهري + فلاتر.
- **`#hr_leaves`:** approval workflow.
- **`#hr_payroll`:** توليد + approval + دفع + قيد GL.

## 4) دورة العمل التفصيلية

**Onboard:** signup_request → users row → hr_employee_profile → position + branch + department → welcome.
**Attendance:** يومي (7 حالات: present/absent/late/leave/holiday/sick/remote).
**Leaves:** `draft → submitted → approved / rejected → cancelled`.
**Payroll:** `draft → approved → paid` + `create_journal_for_payroll`.
**Termination:** `is_active = false` + settlement.
